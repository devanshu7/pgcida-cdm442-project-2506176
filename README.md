# Dynamic Pricing Strategy Simulator
An e-commerce tool to optimize pricing using the Online Retail II dataset. This project calculates price elasticity across categories and simulates 5%, 10%, and 15% price shifts. It features a Power BI dashboard to visualize revenue, margins, and data-driven pricing strategies for better decision-making.
The raw data used for this project is too large for GitHub. You can download it here: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci
